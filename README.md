# ⚡ Real-Time EV Monitoring and Analytics Dashboard (Simulation-Based)

## 📌 Overview
This project presents a software-based simulation of an Electric Vehicle (EV) monitoring system. It provides real-time visualization of key EV parameters such as speed, battery state of charge (SOC), temperature, and estimated driving range.

## 🎯 Objectives
- Simulate real-time EV operational data  
- Visualize parameters using a dashboard interface  
- Implement alert mechanisms for critical conditions  
- Perform data logging and analysis  

---

## 🛠️ Technologies Used
- Python  
- Streamlit  
- Pandas  
- Plotly  

---

## ⚙️ System Architecture
The system is designed using a modular approach:

- **Data Simulation Module** – Generates real-time EV data  
- **Analytics Module** – Computes range and performance metrics  
- **Alert System** – Detects critical conditions  
- **Visualization Layer** – Displays real-time dashboard

## 📊 Results

- Real-time monitoring achieved
